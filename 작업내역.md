## 4차 작업 — 3·4·5단계 전면 개편

### 3단계 공정 문제 분석

**purposeOptions 전면 교체** (인쇄업 실제 문제점 기반)

공정별(절단·커팅 / 인쇄 / 기타) × 목적별(기존장비 교체 / 신규장비 도입) 조합으로 각 8~10개 항목, **"현재 문제 → 개선 목표"** 형식으로 작성:

- 절단/커팅 공정 - 기존장비 교체 (10개): 노후 고장·정밀도 저하·사이즈 부족·두꺼운 소재 불가·속도 저하·수동 보정·데이터 연동 불가·소모품 문제·작업환경·데이터 관리
- 절단/커팅 공정 - 신규장비 도입 (8개): 외주 납기·원가·소량 대응·공정 일원화·샘플 제작·신규 품목·긴급 대응·품질 편차
- 인쇄 공정 - 기존장비 교체 (10개): 고장·헤드막힘·속도·색상편차·특수잉크·사이즈·UV경화·잉크비용·RIP호환·AS불가
- 인쇄 공정 - 신규장비 도입 (10개): 외주 의존·다소재·소량 대응·특수인쇄·샘플 대응·신규 품목·외주비·긴급 납품·커팅 연계·대형 소재
- 기타 - 기존장비 교체 (6개) / 신규장비 도입 (5개)

**"문제 설명 및 개선 목표" 박스 삭제** (텍스트 입력 2개 + 기대효과 체크박스 8개 전체 제거)

## 5차 작업 — 평균매출 판정 로직 수정

**기준:** 소공인 스마트제조 지원사업 평균매출 조건은 **2억원 초과** 기업이 지원 대상

| 조건 | 변경 전 | 변경 후 |
|---|---|---|
| 평균매출 > 2억원 | 초록 "기준 충족 가능" | 초록 **"2억원 초과 — 지원 대상 가능"** |
| 평균매출 ≤ 2억원 | 빨강 "기준 초과 가능성 있음" | 빨강 **"2억원 이하 — 기준 미달 (지원 대상 제외)"** |

판정 로직: `avg <= limit` (이하면 OK) → `avg > limit` (초과해야 OK) 로 수정
